# GrupoTSC

Sitio web institucional desarrollado para **GrupoTSC**, empresa especializada en infraestructura tecnológica, conectividad y seguridad electrónica.

## 🌐 Enlaces

- **Sitio web:** https://slvallejos.github.io/GrupoTSC/
- **Repositorio:** https://github.com/SLVallejos/GrupoTSC

## 🚀 Servicios

- Cableado estructurado
- Redes y conectividad
- Instalación y configuración Wi-Fi
- Cámaras de seguridad CCTV
- Control de acceso
- Servidores y racks
- Infraestructura IT
- Soporte técnico

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages
